# Dane

Ten katalog zawiera dane treningowe i testowe.

## Zawartość

Tutaj znajdziesz:
- Dane treningowe dla modelu
- Dane testowe
- Przykładowe datasety
- Zbiory danych do walidacji

## Uwaga

Duże pliki danych nie powinny być commitowane bezpośrednio do repozytorium. 
Zamiast tego, dodaj je do `.gitignore` i przechowuj w zewnętrznym storage.
